Addons
Standard Procedure uses a few addons in its raiding suite. While we don't have any formal requirements for which addons to use, we highly recommend the following addons:
  • Deadly Boss Mods which enables raid event warnings etc.
  • CEPGP which provides a GUI for our EPGP-system.
  • Details! Damage/Threat Meter which gives you an overview of threat (among other things).
  • WeakAuras 2 does a lot of things, but essentially allows the dispay of highly customizable graphics which indicate buffs, debuffs, and other relevant information.
  • (Healers) LibHealComm-4.0 enables visualization of incoming heals (check supported AddOns).
  • (Paladins) PallyPower makes blessing management easy.
